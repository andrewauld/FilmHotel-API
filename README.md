# FilmHotel API

A Film & TV Discovery API built with **FastAPI** and **PostgreSQL**. Search films via [TMDB](https://www.themoviedb.org/), maintain watchlists, log what you've watched, rate films, explore analytics, and get AI-powered personalised recommendations via [Google Gemini](https://ai.google.dev/).
Deployment can be found [here](https://filmhotel-api.onrender.com).

---

## Quick Start

### Prerequisites

- **Python 3.10+**
- **PostgreSQL** (running locally or remotely)
- A **Google Gemini API key**
- A **TMDB API key**

### 1. Clone & set up

```bash
git clone https://github.com/andrewauld/FilmHotel-API.git
cd FilmHotel-API

# Create a virtual environment and install dependencies
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Configure environment

Set up your environment variables in a `.env` file following the template below. Make sure to include this file in the `.gitignore` file.

```dotenv
# ── Database ──────────────────────────────────────────
DATABASE_URL=postgresql://username:password@localhost:5432/filmhotel

# ── JWT Authentication ────────────────────────────────
SECRET_KEY=change-me-to-a-random-secret
ACCESS_TOKEN_EXPIRE_MINUTES=60

# ── TMDB API ──────────────────────────────────────────
# Get your free API key at: https://www.themoviedb.org/settings/api
TMDB_API_KEY=your-tmdb-api-key

# ── Google Gemini AI ──────────────────────────────────
# Get your API key at: https://aistudio.google.com/apikey
GEMINI_API_KEY=your-gemini-api-key
```

### 3. Set up the database

```bash
# Create the PostgreSQL database
createdb filmhotel

# Run migrations
.venv/bin/python -m alembic upgrade head
```

### 4. Run the server

A convenience script is provided to run the server:

```bash
./run.sh
```

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | FastAPI |
| Database | PostgreSQL |
| ORM | SQLAlchemy 2.0 |
| Migrations | Alembic |
| Auth | JWT (python-jose + passlib/bcrypt) |
| Film Data | TMDB API |
| AI Recommendations | Google Gemini |
| Testing | pytest |

## Project Structure

```

│   ├── main.py          # FastAPI entry point
│   ├── config.py        # Environment settings
│   ├── database.py      # DB engine & session

├── alembic/             # Database migrations
```
